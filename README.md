# Лабораторная работа №1

## Сравнение методов решения СЛАУ


## Цель работы

Реализация и исследование методов решения систем линейных алгебраических уравнений (СЛАУ), а также их сравнительный анализ по времени выполнения и точности.


## Постановка задачи

Необходимо реализовать и исследовать следующие методы:

* метод Гаусса без выбора главного элемента
* метод Гаусса с выбором главного элемента
* метод LU-разложения

Требуется:

* сравнить время работы методов
* оценить точность решений
* исследовать эффективность при нескольких правых частях
* проверить поведение методов на плохо обусловленных матрицах


## Описание реализованных алгоритмов

### Метод Гаусса без выбора главного элемента

Метод основан на приведении матрицы системы к верхнетреугольному виду с помощью элементарных преобразований. После этого выполняется обратная подстановка.

Недостаток метода — численная неустойчивость при малых ведущих элементах.


### Метод Гаусса с выбором главного элемента

На каждом шаге выбирается максимальный по модулю элемент в текущем столбце и производится перестановка строк.

Это позволяет повысить устойчивость метода и уменьшить ошибки округления.


### LU-разложение

Матрица A представляется в виде произведения:

A = L * U

где:

* L — нижнетреугольная матрица
* U — верхнетреугольная матрица

Решение системы Ax = b выполняется в два этапа:

1. Ly = b — прямая подстановка
2. Ux = y — обратная подстановка

Преимущество метода — возможность повторного использования разложения.

### 2. Несколько правых частей

* k = 1, 10, 100

* повторное применение метода Гаусса
* использование LU-разложения

## Результаты

В ходе экспериментов установлено:

* метод Гаусса без выбора главного элемента работает быстрее, но менее устойчив
* метод с выбором главного элемента обеспечивает более точные результаты
* LU-разложение требует дополнительных затрат на этапе разложения
* при увеличении числа правых частей LU-разложение становится эффективнее


## Выводы

1. Метод Гаусса без выбора главного элемента является простым, но численно неустойчивым.
2. Использование выбора главного элемента повышает точность решения.
3. LU-разложение эффективно при решении систем с несколькими правыми частями.
4. Для плохо обусловленных матриц рекомендуется использовать методы с выбором главного элемента.
